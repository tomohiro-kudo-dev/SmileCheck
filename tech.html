SmileCheck 技術解説

SmileCheck は、表情スコアを計測するシンプルなWebアプリです。主に美容部員や接客業の方の「笑顔の整え直し」をサポートする目的で設計されました。

🛠 使用技術

  • HTML / CSS / JavaScript：フロントエンド構成
  • MediaPipe FaceMesh：顔のランドマーク検出
  • localStorage：履歴保存
  • GitHub Pages：ホスティング

📐 判定ロジック

MediaPipeから得た顔のランドマークをもとに、以下の要素をスコア化します：

  • 口角の左右の距離（横幅）
  • 唇の上下の開き具合

これらをしきい値としてスコア（60〜90点）をつけ、リアルタイムに更新。90点以上では「😊 Good Smile!」アニメーションが表示されます。

📊 履歴管理

ブラウザの localStorage に履歴を保存し、過去5件まで表示します（ユーザーごと・ブラウザごとに保存）。
